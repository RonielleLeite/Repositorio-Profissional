# 🎓 Repositório Profissional - Portfólio

Um **portfólio curricular completo** que apresenta meus projetos, habilidades e experiências como desenvolvedor Full Stack. Este repositório funciona como um showcase profissional de trabalhos práticos desenvolvidos com HTML, CSS, JavaScript e frameworks modernos.

---

## 🚀 Funcionalidades

- **Apresentação Profissional**: Seção de boas-vindas com foto e descrição
- **Projetos em Destaque**: Galeria de projetos com descrições e links
- **Habilidades Técnicas**: Listagem de tecnologias e competências
- **Experiência Profissional**: Timeline de experiências e formação
- **Contato Direto**: Formulário e links para redes sociais
- **Design Responsivo**: Totalmente adaptado para mobile, tablet e desktop

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
| :--- | :--- |
| **HTML5** | Estrutura semântica e moderna. |
| **CSS3** | Estilização avançada e layouts responsivos. |
| **JavaScript (ES6+)** | Interatividade e funcionalidades dinâmicas. |
| **Bootstrap** | Framework CSS para design responsivo. |
| **GitHub Pages** | Hospedagem estática gratuita. |

## 📦 Como Executar o Projeto

Para visualizar este portfólio localmente, siga os passos abaixo:

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/RonielleLeite/Repositorio-Profissional.git
   ```

2. **Acesse a pasta do projeto**:
   ```bash
   cd Repositorio-Profissional
   ```

3. **Abra no navegador**:
   - Abra o arquivo `index.html` diretamente no seu navegador, ou
   - Use um servidor local (ex: Live Server no VS Code)
