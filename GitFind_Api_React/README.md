# GitFind - Explorador de Perfis do GitHub

## 📌 Sobre o Projeto
O **GitFind** é uma aplicação web desenvolvida em **React** que consome a **API oficial do GitHub**.  
Ele permite buscar perfis de usuários e listar seus repositórios públicos de forma simples e intuitiva.

## 🚀 Funcionalidades
- Buscar usuários do GitHub pelo nome
- Exibir informações do perfil:
  - Foto de avatar
  - Nome
  - Biografia
- Listar todos os repositórios públicos
- Interface responsiva e amigável

## 🛠️ Tecnologias Utilizadas
- React
- JavaScript (ES6+)
- HTML5
- CSS3
- API do GitHub
