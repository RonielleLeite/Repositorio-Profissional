Gerenciador de Texto com DOM e JavaScript

Este projeto foi desenvolvido com foco em praticar manipulação do DOM utilizando JavaScript puro. A aplicação consiste em uma caixa de texto acompanhada de dois botões principais: Inserir Texto e Limpar Texto. - Ao inserir conteúdo na caixa e clicar em "Inserir Texto", o texto é automaticamente salvo e exibido abaixo, permitindo ao usuário visualizar o histórico de entradas. - Caso o botão seja acionado sem que nenhum texto tenha sido digitado, o sistema apresenta uma mensagem de alerta, garantindo uma experiência mais intuitiva e evitando erros de uso. - O botão "Limpar Texto" remove rapidamente o conteúdo digitado, oferecendo praticidade e controle ao usuário. Este projeto demonstra conceitos fundamentais de interatividade em páginas web, como: - Manipulação de elementos HTML via DOM; - Validação de entrada de dados; - Uso de eventos para dinamizar a experiência do usuário. Além de ser um exercício prático de lógica e programação, o projeto reflete a importância de criar interfaces simples, funcionais e responsivas, que valorizam a usabilidade e a clareza na interação com o usuário.

Caixa de Supermercado – Simulador Interativo

Este projeto foi desenvolvido com o objetivo de treinar JavaScript e manipulação do DOM, criando uma experiência interativa que simula o processo de atendimento em um caixa de supermercado. - O usuário pode adicionar produtos com seus respectivos preços, que são exibidos em uma lista organizada. - O sistema realiza o cálculo automático do valor total, atualizando em tempo real conforme novos itens são inseridos ou removidos. - Inclui funcionalidades de limpar a lista e reiniciar o processo, garantindo praticidade e controle. - Foram aplicados conceitos de validação de entrada, com alerta de erro quando o usuário tenta enviar uma caixa em branco, evitando inconsistências. - Também foi implementada a opção de adicionar desconto, permitindo simular promoções e calcular o valor final com abatimento. - O projeto reforça boas práticas de usabilidade, com mensagens claras e interface simples, tornando a experiência acessível e eficiente. Este simulador demonstra como recursos básicos de JavaScript podem ser aplicados para resolver problemas cotidianos, além de servir como base para sistemas mais complexos de gestão de vendas e controle de estoque. - Detalhe desse projeto é: Quando selecionar a opção de "Alimentação", pedi para adicionar a data de validade do produto.

GitFind - Explorador de Perfis do GitHub

O GitFind é uma aplicação web desenvolvida em React que permite buscar informações detalhadas de perfis do GitHub de forma simples e intuitiva. Utilizando a API oficial do GitHub, o projeto exibe dados como foto de perfil, nome, biografia e lista completa de repositórios públicos do usuário pesquisado.
